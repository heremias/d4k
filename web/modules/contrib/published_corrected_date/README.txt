Published and corrected dates

This module adds 3 propreties to all nodes that cannot be modified in the UI,
but are available for display via Views and Layout Builder.

The fields are:

- Publication date: the first time the node was save with the status set to
published
- Last corrected date: the last time it was saved with the status set to
published
- Number of corrections: the number indicating the number of times it was saved
with the status set to published (not counting the first time)
